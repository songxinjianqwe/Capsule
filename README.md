## Project Structure
Capsule containers a cli and a library, which provider atomic operations of container

## Install
1. `go get "github.com/songxinjianqwe/capsule"`
2. `cd $GOPATH/src/github.com/songxinjianqwe/capsule`
3. `go install`
4. `cd $GOPATH/bin`
5.`./capsule help`
